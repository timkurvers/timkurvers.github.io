## Setup

Install [Node.js] or higher for your platform.

Install dependencies through npm:

```bash
npm install
```

## Running

To spin up a development server on `localhost:8000`, run:

```bash
npm start
```

To specify a different port, use the `PORT` environment variable:

```bash
PORT=3000 npm start
```
